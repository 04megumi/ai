# Alpha-Beta Module (Alpha-Beta 模块)

This module implements the Alpha-Beta Pruning algorithm to improve the efficiency of the minimax search in the game of Go (Gomoku). The Alpha-Beta algorithm is used to explore the game tree, pruning branches that cannot influence the final decision.  
本模块实现了 Alpha-Beta 剪枝算法，以提高在五子棋游戏中 minimax 搜索的效率。Alpha-Beta 算法用于探索游戏树，剪除无法影响最终决策的分支。

## Key Features (主要特点)

- Implements the Alpha-Beta Pruning algorithm to efficiently compute the best move for the AI.  
  实现了 Alpha-Beta 剪枝算法，用于高效计算 AI 的最佳走法。
- Works by reducing the number of game states to explore, improving the performance of the AI.  
  通过减少需要探索的游戏状态数来提高 AI 的性能。
- Suitable for deterministic two-player games like Gomoku, Chess, and Checkers.  
  适用于像五子棋、国际象棋和跳棋这样的确定性双人游戏。
